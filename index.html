Intro to Web Design

Learning to build web applications is an arduous process that requires a sprinkle of intelligence, a splash of determination and a whole lot of patience.

This project is a personal portfolio item that I am using to illustrate my knowledge in designing for the web. I have also given consideration to individuals who have no coding experience~ there has never been a better time to learn how to code! Enjoy the journey with.

Before diving in, lets ensure that our toolbox is properly equipped to handle the challenges ahead. This includes selecting a text editor, installing a local host and using a version control system to maintain code. Lets get started!

Web Dev Setup

Setting up a developer environment is a technical process, that when done right, will give you divine powers. And when done wrong, well....

Choosing a Text Editor is a simple choice with huge ramifications depending on your coding objectives. Most IDEs are suitable for front-end coding projects (ie html, css, javascript). Like buying a car, research what the market has to offer, find what you like, download and install. Fortunately, you won't have to come out of pocket for an IDE as the majority are open-source.

With a text editor installed, you can properly code a web application using industry standard best practices. However, in order to view that which you code, it is imperative to Install a Local Host, which is software that runs locally on a computer and allows developers to test web applications before deployment. The best feature of a local host is that developers can code projects without internet access, yet, still view the results in real-time.

With these tools, you can start coding projects immediately, but having access to a Version Control System will be advantageous in your work as you can manage revisions made to a document throughout its existence, allowing developers to revert a document to a previous state. A VCS is rather complex and requires a bit of studying; and unlike IDE and Local Host software, sooner or later you may have to pay for such services. And it will be worth it!

Coding 101

Coding a web application is analogous to being a polyglot on tour: where you go depends on how well you speak the language. And ones fluency of multiple languages will ultimately dictate the overall experience. Every web developer begins their journey by learning two software programming languages: HTML and CSS.

HTML is the standard markup language used when creating webites semantically. It is used specifically to embed content into a web page using nested Elements, where each element serves a purpose based on the design objective. Enhancement of any element is possible through manipulating Attributes, Classes and Ids. Learn everything you need to know about html here.

CSS is the language used to describe the presentation (design) of a markup language document. This language empowers designers to target HTML Elements by styling the layout, adding color, improving accessibility and much more.
